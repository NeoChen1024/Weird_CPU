Weird CPU assembler syntax
==========================

every field in a line is tab-separated

type	symbol	operand	comment
==================================================
c	Comment
asm     org     $0000	Origin Address
i	r	one	Read from "one"
lb	loop0		Label "loop0"
i	add	one
i	wr	DISP	Write to hex display
i	jcu	loop0
i	hlt
db	one	$01
==================================================
